Basic CSS selectors

Name Syntax Description
Universal Selector *{ } Selects all elements.
Type Selector element { } Selects all elements of a given type.
Class Selector .classname { } Selects all elements with a specific class.
ID Selector #idname { } Selects a single element with a specific ID.
List Selector element, element { } Selects multiple elements.
